# Crysis Warhead - Call of Crysis (Gameplay Tweaks)

This mod adjusts player movement physics and weapon parameters to make the gameplay feel faster, more responsive, and agile (bringing a Call of Duty-style super-soldier feel to Crysis).

## Features

- **Base Walk Speed:** Increased standing normal speed from `1.75` to `2.2` m/s.
- **Base Run Speed:** Increased standing run speed from `4.5` to `6.5` m/s.
- **Zero Recoil & Spread:** Zeroed out all physical recoil, recoil multipliers, and spread on the SMG and the AY69.
- **AY69 Tremor Visual Tweaks:** Set first-person firing animations of the AY69 to `speed="6.0"` to provide a smooth, fast vibration on firing rather than a heavy screen bounce. Ironsight camera shake is set to `0.05` for realistic feedback.
- **Automatic Crouch Toggle:** Automatically sets the `cl_crouchToggle` console variable to `1` when loading maps, making crouch toggle work out-of-the-box.
- **100% Clean:** Derived entirely from the original game assets. It contains no third-party code and has no dependencies on other mods.

## Installation

1. Copy `zzzzz_CallOfCrysisTweaks.pak` into your `...\Crysis Warhead\Game\` folder.
