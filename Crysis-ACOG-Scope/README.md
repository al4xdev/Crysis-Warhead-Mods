# Crysis Remastered - R6S Style ACOG Scope

This mod replaces the default reticle of the Assault Scope accessory with a clean, high-resolution (512x512) Rainbow Six Siege style red ACOG reticle.

## Features

- **R6S ACOG Chevron Reticle:** Features a hollow red chevron pointing up at the exact point of aim for precise targeting.
- **High-Definition (512x512):** Rendered at 512x512 resolution for maximum clarity and sharp rendering on high-resolution displays.
- **Distance Markings:** Includes red and black tick marks aligned vertically for distance compensation, matching the iconic R6S visuals.
- **Clean Override:** Only replaces the crosshair texture, having zero performance impact and preserving all other scope model geometry and lens features.

## Installation

1. Copy `zzzzz_ACOGScope.pak` into your `...\Crysis Remastered\Game\` folder.
2. In-game, equip the **Assault Scope** accessory on any weapon to view the updated ACOG reticle.
